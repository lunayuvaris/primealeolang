# is_prime.aleo

## Build Guide

To compile this Aleo program, run:
```bash
aleo build
```

To run it:
```bash
aleo run mint_prime_token aleo1lr6ap3ed4q823wek3kk9eha3e8e98wv40x578zhl6afmds2twqxs8szyj4 5u32
```
